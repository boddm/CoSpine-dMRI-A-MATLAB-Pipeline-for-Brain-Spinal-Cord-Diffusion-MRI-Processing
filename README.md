# CoSpine-dMRI-A-MATLAB-Pipeline-for-Brain-Spinal-Cord-Diffusion-MRI-Processing
This repository contains the MATLAB source code used in the manuscript   **“Spinothalamic Tract Microstructure as a Common Neural Substrate of Pain Sensitivity Across Modalities.”**   It provides a reproducible pipeline for preprocessing, registration, DTI fitting, and tractography across the brain–spinal cord axis.
